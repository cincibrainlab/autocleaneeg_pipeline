# Migrating from PyQt5 to PySide6

<Steps>

### 1. Replace the Qt binding

The `pyproject.toml` dependency list now requires **PySide6** instead of PyQt5 to ensure wheels are available on Linux.

### 2. Update GUI imports

All GUI code under `src/autoclean/tools` imports classes from `PySide6` modules, removing `PyQt5` specific helpers.

### 3. Adjust documentation build

Sphinx mocks `PySide6` during documentation builds to avoid importing GUI libraries on headless systems.

### 4. Verify the upgrade

```bash
pip install .[gui]
pytest tests/unit -q
```

</Steps>
